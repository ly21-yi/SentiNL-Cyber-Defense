# SentiNL-Cyber-Defense

SentiNL-Cyber-Defense is an AI-powered cyber defense system developed as a **team project during the USF HackJam**.  
The project focuses on automated threat analysis and security insights using large language models and external threat intelligence sources.
---
## Overview
SentiNL-Cyber-Defense integrates AI and external security data to help identify, analyze, and contextualize potential cyber threats.  
It was built collaboratively in a hackathon environment, emphasizing rapid prototyping, modular design, and real-world cybersecurity use cases.
---
## Key Features
- AI-assisted threat analysis
- Integration with external breach intelligence sources
- Modular and extensible Python architecture
- Structured JSON-based data processing
- LLM-powered security insights
---

- USF HackJam team project
